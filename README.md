# journal-tracker-101
Simple journal writing tool for tracking daily personal development. 

## Usage
Write, edit, delete notes every day. 

## Tools used:
* Java 15
* SQLite 3.34.0
* JOOQ 3.14.9
* Maven
* NetBeans
